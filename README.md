# Andiamo Events - Tunisia's Premier Nightlife Experience

A modern, responsive website for Andiamo Events, showcasing Tunisia's premier nightlife events and experiences.

## 🚀 Features

- **Responsive Design** - Works perfectly on all devices
- **Multi-language Support** - English and French
- **Dynamic Content** - Editable from Supabase dashboard
- **Modern UI** - Beautiful gradients and animations
- **Event Management** - Gallery, events, and ambassador program

## 🛠️ Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **Backend:** Supabase (Database + Storage)
- **Deployment:** Vercel/Netlify ready

## 📦 Installation

```bash
npm install
npm run dev
```
